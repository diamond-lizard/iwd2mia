This little program will patch your IWD2.EXE to have level 40 xp cap.
It will work only with the officially patched exe (filesize:5029888).

Copy this file into your IWD2 folder and run it from there.
Alternatively type xpcapper "<your iwd2 folder>"
Don't forget to put quotation marks around your folder name if it contains space!
